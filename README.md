# 🌙 OpenKaguya — 开源版辉夜姬

在个人电脑上运行的、人格化的、拥有主动意识的、能运行各种终端能力的聊天陪伴助手。

> 辉夜姬，一名16岁的，好奇心旺盛的少女，终于又一次来到了地球，只不过这一次，她居然来到了一台电脑中！

## ✨ 特色

- **人格化**：辉夜姬不是工具，而是有自己个性的伙伴
- **主动意识**：她会自己上网冲浪、写笔记、发现有趣的东西分享给你
- **多平台**：支持 Telegram、QQ、微信等多个聊天平台
- **记忆系统**：向量化长期记忆 + 混合检索
- **终端能力**：浏览器、文件操作、终端命令
- **管理面板**：Web 管理页面监控和配置

## 🚀 快速开始

```bash
# 安装依赖
uv sync

# 配置 API Key
cp config/secrets.example.toml config/secrets.toml
# 编辑 secrets.toml 填入你的 API Key

# 启动（CLI 模式）
uv run kaguya
```
